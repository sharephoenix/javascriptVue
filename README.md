## vue 创建命令
# 电脑环境搭建

1. 安装 node：
    brew install node
2. 安装 vue：
    npm install vue
3. 安装 脚手架：
    npm install -g vue-cli
4. 初始化项目命令
    vue init webpack my-vue-component
5. 项目安装依赖，
    npm install
6. 启动项目
    npm run dev
